# Sources

Source cards point to raw material and summarize why it matters. Do not copy long excerpts.

## Source Card Template

| Field | Value |
| --- | --- |
| Source | path or URL |
| Type | code, test, doc, issue, article, release note, conversation, other |
| Stability | stable, medium, volatile |
| Extracted knowledge | short summary |
| Linked pages | wiki links |
| Last reviewed | YYYY-MM-DD |
