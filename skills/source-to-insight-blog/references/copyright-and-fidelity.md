# Copyright And Fidelity Boundary

This skill optimizes for source fidelity without turning a third-party article into a public mirror.

## Allowed Default

- High-fidelity Chinese reconstruction or annotated digest.
- Short quotations within applicable limits.
- Tables rebuilt from facts when attribution and links are clear.
- Media embeds or links using the original source URL and alt/caption metadata.
- Source ledger entries that identify claims, links, media, and access limits.

## Not Allowed By Default

- Full article repost.
- Full line-by-line translation of a third-party article.
- Large contiguous excerpts copied into public files.
- Downloading and republishing media when the license is unclear and linking would satisfy the need.

## When The User Demands Full Translation

Explain the boundary once, then produce the closest useful compliant artifact:

- section-by-section Chinese reconstruction;
- preserved structure, facts, numbers, references, figures, footnotes, and glossary;
- a link to the original for exact text;
- an explicit note that the public artifact is not a substitute for the original wording.

Do not hide this limitation in the source ledger only; the public post should make the boundary visible.
