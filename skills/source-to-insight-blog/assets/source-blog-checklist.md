# Source Blog Checklist

- [ ] Canonical source URL, title, author, publish date, and access date captured.
- [ ] Access limitations recorded.
- [ ] Source headings and order preserved.
- [ ] Media ledger includes URL, type, alt text, caption, and placement.
- [ ] Important links and named references are preserved.
- [ ] Tables, footnotes, and glossary entries are represented.
- [ ] Chinese source-fidelity layer appears before summary/visualization.
- [ ] Copyright boundary appears in the public post.
- [ ] Effective-interact summary or visualization layer exists and is secondary.
- [ ] Project-iteration review says now / later / not now.
- [ ] Rendered screenshot confirms the blog is readable.
- [ ] `validate-source-blog.mjs <post-dir>` passes.
- [ ] Insight site validation passes.
